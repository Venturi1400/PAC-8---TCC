Gestão de Equipamentos de Campo – WEG Tintas
Sobre o Projeto

Este projeto foi desenvolvido como proposta de Trabalho de Conclusão de Curso (TCC) do curso de Engenharia de Software do Centro Universitário Católica de Santa Catarina. A proposta consiste na criação de um aplicativo móvel voltado para o controle e gerenciamento dos equipamentos utilizados pelos técnicos de campo do setor de Serviço ao Cliente (SAC) da WEG Tintas.

Atualmente, o acompanhamento desses equipamentos é realizado por meio de processos manuais, utilizando relatórios e registros fotográficos, o que dificulta o controle das informações, aumenta o risco de perda de dados e torna o acompanhamento das manutenções preventivas menos eficiente. Diante desse cenário, o projeto busca desenvolver uma solução que centralize todas as informações em uma única plataforma, proporcionando maior organização, rastreabilidade e confiabilidade dos registros.

O aplicativo permitirá que os equipamentos sejam cadastrados e vinculados aos respectivos técnicos responsáveis. Além disso, os usuários deverão realizar validações periódicas dos equipamentos por meio do preenchimento de informações e do envio de registros fotográficos, criando um histórico detalhado das condições de utilização de cada item. O sistema também enviará notificações automáticas relacionadas às manutenções preventivas, contribuindo para a redução de atrasos e falhas operacionais.

A solução contará com três perfis de acesso. O administrador será responsável pela gestão geral do sistema, incluindo o gerenciamento de usuários e o acompanhamento dos relatórios gerenciais. O coordenador atuará como principal responsável pelo controle operacional da plataforma, realizando cadastros, acompanhando validações e emitindo relatórios. Já o técnico terá acesso apenas aos equipamentos sob sua responsabilidade, podendo realizar validações e acompanhar informações relacionadas às manutenções.

O desenvolvimento será realizado utilizando Flutter, permitindo a criação de uma aplicação compatível com dispositivos Android e iOS a partir de uma única base de código. Para o gerenciamento dos dados serão utilizados os serviços da plataforma Firebase, incluindo autenticação de usuários, banco de dados em nuvem e armazenamento de imagens, garantindo segurança, sincronização em tempo real e facilidade de manutenção.

Como resultados esperados, o projeto pretende reduzir a dependência de controles manuais, aumentar a confiabilidade das informações, melhorar o acompanhamento das manutenções preventivas e fornecer relatórios gerenciais que auxiliem gestores e coordenadores na tomada de decisões. Dessa forma, espera-se contribuir para a otimização dos processos internos do setor de SAC da WEG Tintas e para o aumento da eficiência operacional da equipe.

O projeto foi desenvolvido com finalidade acadêmica como parte das atividades do Trabalho de Conclusão de Curso, podendo futuramente ser expandido com novas funcionalidades e integrações de acordo com as necessidades da organização.
